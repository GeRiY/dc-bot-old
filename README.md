## Requirements

1."discord.js": "^11.5.1",
2."dotenv": "^8.2.0",
3."request": "^2.88.2",
4."sync-request": "^6.1.0"

## Installation Steps (if applicable)

1. Clone repo
2. Run `npm install`
3. Add Discord credentials in a `.env` file
3. Run `node index.js`
4. Interact with your Discord bot via your web browser
